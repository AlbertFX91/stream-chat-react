## How-to Guide for Customizing the ChannelSearch Search Function

`ChannelSearch` is a UI component that searches for chat users. It can be used standalone or used in the `ChannelList` via the `showChannelSearch` prop (default is false).
The input searches for users by default, but the `searchFunction` prop can be used to override this out of the box behavior.
Selection of a search result will set the active channel with the selected user.
