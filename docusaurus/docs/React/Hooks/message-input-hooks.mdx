The `MessageInput` component provides and uses custom hooks to assemble the [`MessageInputContext`](../contexts/message-input-context.mdx).

Below is an outline of the two main exposed hooks, [`useMessageInputState`](./message_input_hooks#usemessageinputstate)), as well as the smaller exposed hooks, grouped by function, that are used within, as well as [`useCooldownTimer`](./message_input_hooks#usecooldowntimer)).

Also, included are the hooks used by the [`DefaultTriggerProvider`](../message-input-components/ui_components#defaulttriggerprovider.mdx). All provided hooks can be useful when building a custom `MessageInput` component.
